    DatumWeb.SearchLive (DeepLynx v0.0.1)

    This is our tab for running searches across the entirety of the Datum ecosystem. If state exists, only hold the state of the query itself and filters.

    This should not embed any additional live views - only components. If you need to build a live view, let's say a FileViewLive or something, then you should message the parent to spawn the relevant tab - not spawn and save it
